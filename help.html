Welcome to the Help Page!

A quick guide on how to use this tool effectively.

A task has a link, name, and date attached to it. The link for a task is directly embedded in the name for easy acess.

  • The bolded number in parentheses next to each task shows how many days are remaining until that task is due.
  • The red "X" button next to every task will remove that task when it is clicked.
  • The "Edit" button allows you to edit the details of an assignment.
  • Use the backspace key while in the "link" box to delete the URL completely.
  • When the extension is opened, the link box is automatically filled with the URL of the currently active tab.
  • Use the Spacebar in the "link" box to autofill the URL of the currently active tab.
  • The keyboard shortcut Ctrl (Cmd) + Shift + A will open/close the extension.
  • Press Enter while typing a name for the task (in the "name" box) to finish the task creation.
  • You can also finish task creation by clicking the green "Finish" button.
